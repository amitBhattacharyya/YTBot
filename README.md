README – YouTube Transcript Chat App
Overview

A lightweight  app that:
    Fetches the transcript of  public YouTube video.
    Indexes the transcript for quick lookup.
    Provides a chat interface where users can ask questions or discuss topics from the video, with AI‑generated answers based on the transcript content.

Features

    One‑click transcript retrieval via YouTube’s caption API (or yt-dlp fallback).
    Local vector store (e.g., faiss or chromadb) for semantic search.
